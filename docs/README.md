# Navigation 封装文档索引

本文档集用于规划本地开发机上的导航联动重写与封装。当前阶段只做设计，不直接改动 `BotDog-jetson-main`、`Superlio-main`、`dddmr_global_planner-main` 的源码。

核心原则：

- 先在本地原生 ROS2 Humble 环境调试，跑通后再 Docker 化。
- Navigation 自身保持低耦合，BotDog 只作为一个参考接入方。
- 对外提供稳定 ROS2/CLI/API 边界，具体前后端不得反向污染 Navigation 核心。
- 不重写 Super-LIO、terrain_analysis、global_planner 的核心算法。
- 把硬编码路径、地图文件、雷达网络、topic/frame 统一配置化。
- 任何封装都必须支持后续迁移到 Jetson ARM64。
