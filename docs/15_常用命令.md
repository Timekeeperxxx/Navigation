## 使用 MID360 ROS2 bag 测试建图

`/home/frankluo/Projects/mid360_raw_01` 是 ROS2 bag，包含：

- `/livox/lidar`: `livox_ros_driver2/msg/CustomMsg`
- `/livox/imu`: `sensor_msgs/msg/Imu`

用 bag 测试时不要启动真实 Livox driver，所以 `launch_livox:=false`。

终端 1：启动 Navigation 建图。

```bash
cd /home/frankluo/Projects/Navigation
source /opt/ros/humble/setup.bash
source install/setup.bash

MAP_DIR=/home/frankluo/Projects/Navigation/maps/bag_mid360_raw_01_$(date +%Y%m%d_%H%M%S)

ros2 launch nav_bringup mapping.launch.py \
  map_dir:=$MAP_DIR \
  launch_livox:=false \
  launch_lio:=true \
  launch_terrain:=true \
  rviz:=true
```

终端 2：播放 bag。

```bash
cd /home/frankluo/Projects/Navigation
source /opt/ros/humble/setup.bash
source install/setup.bash

FASTDDS_BUILTIN_TRANSPORTS=UDPv4 ros2 bag play /home/frankluo/Projects/mid360_raw_01 \
  --read-ahead-queue-size 1000
```

bag 播完后，在终端 1 按 `Ctrl+C`，会触发保存：

- `map.pcd`
- `terrain_map_xxx_ground.pcd`
- `terrain_map_xxx_base_footprint_fill.pcd`

中途手动保存 terrain：

```bash
ros2 service call /save_terrain_map std_srvs/srv/Trigger "{}"
```
