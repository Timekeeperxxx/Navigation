# 06 配置与地图格式

## 配置目标

所有随机器、网络、路径变化的参数都必须外置，避免写死在 launch、C++ 或 Python 里。

## 推荐配置文件

```text
Navigation/config/
  robot_nav.yaml
  livox_mid360.yaml
  mapping.yaml
  navigation.yaml
  global_planner.yaml
```

第一阶段可以先只写 YAML 和 launch 参数，不引入数据库。

## 地图目录格式

推荐每个场景一个目录：

```text
maps/
  Scene001_warehouse/
    map.pcd
    ground.pcd
    footprint_fill.pcd
    metadata.yaml
    waypoints.json
```

兼容旧命名：

```text
*_map.pcd
*_ground.pcd
*_footprint_fill.pcd
```

## `metadata.yaml`

推荐字段：

```yaml
schema_version: 1
scene_id: Scene001_warehouse
created_at: "2026-07-03T00:00:00Z"
frame_id: map
base_frame: base_footprint
lidar:
  model: Livox MID360
  lidar_ip: 192.168.123.179
files:
  map: map.pcd
  ground: ground.pcd
  footprint_fill: footprint_fill.pcd
source:
  lio: super_lio
  terrain: terrain_analysis
  planner: global_planner
notes: ""
```

## 文件职责

| 文件 | 用途 |
|---|---|
| `map.pcd` | Super-LIO relocation 加载 |
| `ground.pcd` | global_planner 静态地面图 |
| `footprint_fill.pcd` | global_planner planground 辅助规划层 |
| `waypoints.json` | 可选，前端/后端点位导出 |
| `metadata.yaml` | 地图元数据和兼容信息 |
