# 17 Go2/B2 控制切换

## 目标

Navigation 的规划、避障和任务层不绑定具体机器狗型号。控制目标通过参数切换：

```text
Navigation
  -> /cmd_vel
  -> /cmd_vel_safe
  -> nav_robot_control
  -> Go2 WebRTC 或 B2 控制桥
```

默认不启动底盘控制，避免离线调试时误发速度。

## 新增包

`nav_robot_control` 提供两个适配器：

| 适配器 | 启动参数 | 作用 |
|---|---|---|
| `go2_webrtc_bridge` | `robot_model:=go2_webrtc` 或 `robot_model:=go2` | 订阅 `/cmd_vel_safe`，通过 `unitree_webrtc_connect` 下发 Go2 Sport `Move`/`StopMove` |
| `b2_cmd_vel_bridge` | `robot_model:=b2_sdk`、`robot_model:=b2_topic` 或 `robot_model:=b2` | 订阅 `/cmd_vel_safe`，限幅后发布到可配置 B2 SDK 桥接输入 topic |

B2 当前按 topic bridge 接入。真实 Unitree SDK2 节点可以订阅 `b2_cmd_vel_topic`，后续也可以在该适配器内部替换为直接 SDK2 调用，但不需要改 Navigation core。

## 通用启动参数

| 参数 | 默认值 | 说明 |
|---|---|---|
| `enable_robot_control` | `false` | 是否启动底盘控制适配器 |
| `robot_model` | `none` | `go2_webrtc`/`go2`、`b2_sdk`/`b2_topic`/`b2` |
| `robot_cmd_vel_topic` | `/cmd_vel_safe` | 适配器订阅的速度话题 |
| `robot_nav_stop_topic` | `/nav_stop` | 停止/急停话题 |
| `robot_nav_start_topic` | `/nav_start` | 可选启动门控话题 |
| `robot_status_topic` | `/robot_control/status` | 适配器状态 JSON |

共用安全参数在 `src/nav_robot_control/config/robot_control.yaml`：

```yaml
watchdog_timeout_sec: 0.3
max_linear_x: 0.15
max_linear_y: 0.10
max_angular_z: 0.40
```

## Go2 WebRTC

先安装并单独验证 WebRTC 连接：

```bash
cd /home/frankluo/Projects/unitree_webrtc_connect-master
python3 -m pip install --user .
```

Go2 固件 `1.1.15+` 可能需要 AES key：

```bash
unitree-fetch-aes-key --email you@example.com --password '...' --device-type Go2 --region cn
```

启动 Navigation 并控制 Go2：

```bash
ros2 launch nav_bringup navigation.launch.py \
  scene_dir:=/path/to/scene \
  enable_path_follower:=true \
  enable_dynamic_avoidance:=true \
  enable_robot_control:=true \
  robot_model:=go2_webrtc \
  go2_connection_method:=LocalAP
```

当前 Go2 真机已验证的控制序列是：

```text
保持当前 motion mode
  -> BalanceStand
  -> no-reply Sport/MCF Move
```

对应默认参数：

```text
go2_ensure_motion_mode:=false
go2_stand_command:=BalanceStand
go2_move_command_mode:=no_reply
go2_use_remote_command_from_api:=false
```

如果是 STA 模式或新固件需要 AES key，可额外指定：

```text
go2_connection_method:=LocalSTA
go2_ip:=192.168.8.181
go2_aes_128_key:=<32-hex-chars>
```

Go2 bridge 行为：

- `/cmd_vel_safe.linear.x` -> Go2 Sport/MCF `Move.x`
- `/cmd_vel_safe.linear.y` -> Go2 Sport/MCF `Move.y`
- `/cmd_vel_safe.angular.z` -> Go2 Sport/MCF `Move.z`
- `/nav_stop=true`、节点退出 -> Go2 `StopMove`
- 普通零速度和 watchdog 软停 -> Go2 零速度命令

可选打开 Go2 自带避障：

```yaml
go2_webrtc_bridge:
  ros__parameters:
    enable_builtin_obstacle_avoidance: true
```

该能力只作为底盘最后一层安全保护，不替代 Navigation 的 `/nav/local_obstacles` 动态避障。

## B2 控制桥

当前 B2 适配器输出标准 `geometry_msgs/msg/Twist` 到外部 B2 控制桥：

```bash
ros2 launch nav_bringup navigation.launch.py \
  scene_dir:=/path/to/scene \
  enable_path_follower:=true \
  enable_dynamic_avoidance:=true \
  enable_robot_control:=true \
  robot_model:=b2 \
  b2_cmd_vel_topic:=/unitree/b2/cmd_vel
```

外部 B2 SDK2 bridge 需要订阅：

```text
/unitree/b2/cmd_vel
```

并把 Twist 转成 B2 实际运动指令。Navigation 侧只保证限幅、watchdog 和 `/nav_stop` 停车输出。

## 切换方式

Go2：

```bash
robot_model:=go2_webrtc
```

B2：

```bash
robot_model:=b2
```

除 `robot_model` 和机型连接参数外，建图、重定位、规划、动态避障、路径跟随参数保持一致。

## 验收顺序

1. 不接真机，只启动 `robot_model:=b2`，确认 `/unitree/b2/cmd_vel` 有零速 watchdog 和限幅后的 Twist。
2. Go2 单独验证 WebRTC 连接、小速度、`StopMove`。
3. Navigation 离线路径跟随输出 `/cmd_vel_safe`，确认 adapter 状态为 `commanding`。
4. 真机架空或安全区域低速测试 `/nav_stop`。
5. 再接完整定位、路径规划、动态避障。
